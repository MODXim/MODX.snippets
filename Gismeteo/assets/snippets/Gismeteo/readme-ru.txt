==Для установки сниппета

Войдите в панель администратора.
Перейдите в "Элементы - Управление элементами - Сниппеты - Новый сниппет".
Задайте название сниппета (я назвал его Gismeteo), его категорию.
Вставьте код, содержащийся в файле MODx-Gismeteo.php в поле "Код сниппета" без изменений.
Сохраните сниппет.

На сервере создайте директорию
/assets/snippets/gismeteo
Положите туда папки с темами оформления и файлы translate.

Должны существовать пути вроде:
/assets/snippets/gismeteo/НАЗВАНИЕ ТЕМЫ/theme.php
/assets/snippets/gismeteo/translate-ru.inc.php

==Использование сниппета

Откройте страницу, чанк или шаблон в редакторе.

Напишите в месте, где должен быть выведен информер:
[[Gismeteo? &region=`27612`]]
Посмотрите результат на сайте. Если вы все сделали правильно - вы увидите информер погоды для Москвы.

Чтобы получить информер для другого города - воспользуйтесь службой XML Gismeteo по адресу
http://informer.gismeteo.ru/getcode/xml.php
Выберите нужный населенный пункт.
Код вашего города - это часть адреса xml, который указан на странице.
http://informer.gismeteo.ru/xml/!!!!!!27612!!!!!!!_1.xml

=Параметры

region — Код населенного пункта в базе Gismeteo.

theme - Чувствительное к регистру название темы информера. В данной поставке есть 4 темы. Silk, Tango, FarmFresh и Text. Если тема не указана - выбирается тема по-умолчанию - Silk.

tpl — имя чанка шаблона. Если шаблон не передан — будет использоваться шаблон по-умолчанию. не все темы поддерживают шаблоны. например, тема Silk шаблоны не поддерживает. Но вы можете изменить саму тему или управлять отображением через CSS.

Каждая тема поддерживает свои плейсхолдеры, они описаны в файлах
/assets/snippets/gismeteo/НАЗВАНИЕ ТЕМЫ/theme.php

Для вывода полного списка плейсхолдеров, можно в файле theme.php перед парсингом шаблона, вызвать dumper($param). Это выведет дамп пар имя плейсхолдера - его значение на данный момент.

lang - язык вывода. По-умолчанию - 'ru'. Обозначение языка должно совпадать с частью имени файла перевода. translate-ЯЗЫК.inc.php.

==информация

Сниппет кэширует XML. Он обновляется 4 раза в сутки, поэтому сниппет не лазит за новым XML чаще. Время, когда уже пора обновить XML вычисляется автоматически. Кэш xml хранится в папке /assets/cache/ в файлах вида gismeteo870.xml.

Все файлы нужно сохранять в utf-8. Если ваш сайт работает не в utf-8 и это указано в настройках системы MODx - сниппет перекодирует вывод из utf-8 в вашу кодировку.
